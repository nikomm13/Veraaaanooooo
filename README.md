# Summer research about clustering

In this research it explores a real data and apply a machine learning technique of non supervized learning. 
    technique that seeks to group similar objects into sets called clusters, so that
     the elements within a group are as similar as possible 
     to one another and as different as possible from others elements in the groups, similar to a classical partition 
     in set theory. To identify these groups within the data, Are we going to implement two algorithms
    called K-means and DBSCAN, both implemented using three different metrics to compare their effectiveness:
    (Euclidean, DTW, and MSM).
